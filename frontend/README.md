# Crypto News Aggregator

A full-stack crypto news aggregation app built with Vue 3 and Python.

This project scrapes live crypto news headlines using Python and displays them in a modern Vue frontend.

---

## Features

- Live crypto news scraping
- Vue 3 frontend
- FastAPI backend
- Responsive modern UI
- API communication with Axios
- CSV-ready backend data

---

## Tech Stack

### Frontend
- Vue 3
- Vite
- Axios

### Backend
- Python
- FastAPI
- BeautifulSoup
- requests
